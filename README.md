# Farmacia Web

Esta aplicacion muestra una interfaz sencilla para gestionar productos de una farmacia usando Gradio y una base de datos SQLite. Puedes añadir productos, actualizar su cantidad y buscar por nombre.

## Caracteristicas

- Listado de productos con precio y cantidad.
- Añadir nuevos productos.
- Actualizar la cantidad de un producto existente.
- Buscar productos por nombre.
